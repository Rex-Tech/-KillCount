#KillCount
##记录你的人头数

#功能
- 记录玩家人头数
- 人头数排行榜[√]
- 还有更多

#未来展望

>粒子效果

#指令
|Command     |Description       |
|------------|------------------|
|/mykc       |查询你的人头数      |
|/kc [player]|查询[player]的人头数|
|/setkillmoney|set the number of money that players can get when they kill others|
